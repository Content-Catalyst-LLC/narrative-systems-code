# Narrative Systems Code

Open computational companion repository for the Storytelling knowledge series.

This repository supports articles on storytelling, poetics, rhetoric, oral tradition, folklore, mythology, narratology, narrative identity, Joseph Campbell, the monomyth, the hero's journey, Maureen Murdock, the heroine's journey, alternative story structures, media adaptation, public narrative, ethics of representation, and narrative systems.

## Repository Structure

- `articles/storytelling-narrative-meaning-and-structure/`
  - `python/` — narrative arc, character network, motif inventory, and story metadata examples
  - `r/` — narrative tension curves, story metadata summaries, and motif distributions
  - `sql/` — schemas for stories, events, characters, motifs, conflicts, turning points, media forms, and ethical cautions
  - `docs/` — article notes and modeling principles
  - `data/` — synthetic datasets
  - `outputs/` — generated outputs
  - `notebooks/` — notebook placeholders

- `shared/`
  - `schemas/` — reusable narrative-system schemas
  - `narrative-templates/` — story structure and article templates
  - `story-archetypes/` — reusable archetype and structural pattern notes
  - `character-models/` — character-arc templates
  - `plot-structures/` — plot, conflict, tension, and transformation models
  - `rhetorical-frameworks/` — rhetorical move and public narrative templates
  - `cultural-memory/` — oral tradition and memory documentation templates
  - `governance/` — ethical and interpretive cautions
  - `tests/` — future validation tests

## Modeling Principles

1. Do not confuse narrative structure with narrative meaning.
2. Treat computational models as aids to interpretation, not replacements for reading.
3. Preserve cultural and historical specificity.
4. Distinguish motif frequency from symbolic significance.
5. Track ethical risks in representation and public narrative.
6. Treat oral tradition as performance and living transmission, not merely text.
7. Use metadata to make narrative systems visible and revisable.
8. Keep universal story models open to critique.
