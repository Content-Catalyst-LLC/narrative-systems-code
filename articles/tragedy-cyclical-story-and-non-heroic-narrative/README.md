# Tragedy, Cyclical Story, and Non-Heroic Narrative

This companion directory supports the article **Tragedy, Cyclical Story, and Non-Heroic Narrative**.

The scaffold treats tragic, cyclical, and non-heroic narrative as auditable interpretive models. It includes article-specific diagnostics for tragic structure, cyclical structure, non-heroic agency, heroic-overfit risk, review readiness, institutional tragedy, ecological cycles, care narratives, witness structures, refusal, survival, and AI-mediated heroic template forcing.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/non_heroic_narrative_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable non-heroic narrative review templates

## Quick Start

```bash
python3 python/run_non_heroic_narrative_canvas_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_non_heroic_narrative_workflows.R
```
