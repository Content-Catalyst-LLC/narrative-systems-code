# Myths, Legends, Folktales, and Epics

This companion directory supports the article **"Myths, Legends, Folktales, and Epics."**

The workflows operationalize traditional narrative-form analysis as a reviewable storytelling system. They include form-classification scoring, truth-claim audits, cultural-memory mapping, genre-boundary diagnostics, hybrid-form tracking, adaptation-risk scoring, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Myths, legends, folktales, and epics are related but distinct traditional narrative forms. They differ by truth claim, social function, memory orientation, performance context, authority, structure, transmission, and adaptation risk. These categories are useful diagnostic lenses, not rigid containers.

## Quick Start

```bash
python3 python/run_traditional_narrative_forms_canvas_audit.py
Rscript r/run_all_traditional_narrative_forms_workflows.R
```

Outputs are written to `outputs/tables/`, `outputs/json/`, `outputs/markdown/`, and `outputs/figures/`.
