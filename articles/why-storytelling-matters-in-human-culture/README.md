# Why Storytelling Matters in Human Culture

This companion directory supports the article **"Why Storytelling Matters in Human Culture."**

The workflows operationalize storytelling as a cultural system for memory, teaching, identity, belonging, moral imagination, social coordination, public meaning, and ethical governance. They include cultural story function audits, transmission scoring, source transparency checks, representation-risk analysis, public-impact diagnostics, narrative-governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Storytelling matters because human culture is not sustained by information alone. Stories help communities remember, teach, belong, judge, coordinate, mourn, hope, contest power, and imagine futures. A useful cultural-story analysis must examine memory, teaching, identity, belonging, moral imagination, social coordination, cultural transmission, public impact, and ethical risk.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Cultural story function datasets
- Memory, teaching, identity, belonging, and ethics tables
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for cultural storytelling analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for cultural memory, story ethics, and responsible interpretation

## Quick Start

From this article directory:

```bash
python3 python/run_cultural_story_canvas_audit.py
Rscript r/run_all_storytelling_culture_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
