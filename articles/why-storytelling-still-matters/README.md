# Why Storytelling Still Matters

This companion directory supports the article **Why Storytelling Still Matters**.

This final Storytelling companion treats storytelling value as a governance problem: stories should clarify meaning, preserve memory, support public reasoning, protect dignity, carry culture responsibly, disclose uncertainty, preserve human judgment, and remain accountable under digital platform and AI conditions.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/storytelling_value_governance_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable responsible storytelling review templates

## Quick Start

```bash
python3 python/run_storytelling_value_governance_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_storytelling_value_governance_workflows.R
```
