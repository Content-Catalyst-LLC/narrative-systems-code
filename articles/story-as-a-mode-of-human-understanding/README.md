# Story as a Mode of Human Understanding

This companion directory supports the article **"Story as a Mode of Human Understanding."**

The workflows operationalize story as an interpretive structure for human sensemaking. They include sequence audits, causal-framing diagnostics, agency mapping, memory integration, moral-understanding scores, possible-world modeling, overreach-risk scoring, narrative-governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Story helps humans understand experience by connecting events, time, causality, memory, identity, moral judgment, and possible futures. It clarifies experience by giving it interpretable structure, but it can also distort understanding when it imposes false coherence, unsupported causality, selection bias, or premature closure.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Narrative-understanding datasets
- Event sequence, causal frame, agency map, moral understanding, and overreach-risk tables
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for narrative sensemaking analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for sensemaking, narrative overreach, moral interpretation, possible worlds, and responsible narrative analysis

## Quick Start

From this article directory:

```bash
python3 python/run_narrative_understanding_canvas_audit.py
Rscript r/run_all_narrative_understanding_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
