# Law, Evidence, and Narrative Responsibility

This companion directory supports the article **Law, Evidence, and Narrative Responsibility**.

The scaffold treats legal storytelling as an evidence-bound governance problem: legal narratives are tested against evidence support, narrative-overreach risk, procedural voice, testimony responsibility, legal-memory accountability, and AI-assisted legal-narrative risk.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/legal_narrative_responsibility_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable legal narrative review templates

## Quick Start

```bash
python3 python/run_legal_narrative_responsibility_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_legal_narrative_responsibility_workflows.R
```
