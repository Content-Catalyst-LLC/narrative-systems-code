# Gender, Critique, and the Limits of Universal Story Models

This companion directory supports the article **Gender, Critique, and the Limits of Universal Story Models**.

The scaffold treats universal story models as auditable interpretive tools. It includes article-specific diagnostics for universal model fit, universalism risk, critique readiness, alternative-structure signal, gender binary pressure, cultural flattening, intersectional erasure, queer and trans narrative pressure, archive bias, local context, and AI-mediated template governance, plus the shared advanced Catalyst Canvas Python layer.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/universal_story_model_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable story-model critique templates

## Quick Start

```bash
python3 python/run_universal_story_model_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_universal_story_model_workflows.R
```
