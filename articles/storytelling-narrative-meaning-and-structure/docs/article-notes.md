# Storytelling: Narrative, Meaning, and Structure

This folder supports the Storytelling pillar with reproducible examples for:

- narrative arc modeling
- character transformation
- motif inventory
- rhetorical move tracking
- conflict and turning-point metadata
- character relationship networks
- story structure schemas
- oral tradition documentation templates
- ethical cautions for representation

Important distinction: storytelling is not only plot mechanics. It is a system for organizing meaning, memory, identity, conflict, persuasion, cultural transmission, and transformation over time.
