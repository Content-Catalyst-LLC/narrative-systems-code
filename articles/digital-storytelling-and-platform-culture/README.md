# Digital Storytelling and Platform Culture

This companion directory supports the article **Digital Storytelling and Platform Culture**.

The scaffold treats digital storytelling as a platform-governed narrative system: platform narrative integrity, context-collapse risk, formula drift, metrics pressure, archive and memory strength, creator-labor pressure, moderation exposure, and AI synthetic-story risk are tested for coherence, provenance, consent, and human review.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/digital_storytelling_governance_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable platform narrative review templates

## Quick Start

```bash
python3 python/run_digital_storytelling_governance_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_digital_storytelling_governance_workflows.R
```
