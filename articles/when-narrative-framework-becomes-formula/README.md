# When Narrative Framework Becomes Formula

This companion directory supports the article **When Narrative Framework Becomes Formula**.

It treats narrative framework use as a governance problem: when does a useful lens become a rigid formula? The scaffold includes diagnostics for formula drift, framework health, narrative specificity, AI-template risk, heroic overreach, closure pressure, beat rigidity, market-story pressure, and editorial review.

## Quick Start

```bash
python3 python/run_narrative_formula_drift_audit.py --article-root .
python3 python/run_catalyst_canvas_audit.py --article-root . --strict
Rscript r/run_all_narrative_formula_drift_workflows.R
```

## Included

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/narrative_formula_drift_canvas/` layer
- Typed models, scoring, validation, governance notes, exporters, and CLI runners
- CSV, JSON, markdown, R, SQL, HTML, CSS, PHP, Java, docs, shared templates, and tests
