# Storytelling as Intangible Cultural Heritage

This companion directory supports the article **"Storytelling as Intangible Cultural Heritage."**

The workflows operationalize storytelling heritage as a living-practice system. They include living-continuity scoring, safeguarding-readiness diagnostics, heritage-context preservation, transmission-pathway review, consent and access controls, archive-risk scoring, community-governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Storytelling as intangible cultural heritage should be safeguarded as living practice, not merely preserved as text, recording, or archive content. Responsible safeguarding supports transmission, language, performance context, consent, access governance, community authority, adaptation review, and intergenerational continuity.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Storytelling heritage, safeguarding context, transmission pathway, consent/access, archive-risk, and governance datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for storytelling heritage safeguarding and governance analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for living practice, safeguarding, archive risk, consent, access, digital context collapse, and responsible use

## Quick Start

From this article directory:

```bash
python3 python/run_storytelling_heritage_canvas_audit.py
Rscript r/run_all_storytelling_heritage_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
