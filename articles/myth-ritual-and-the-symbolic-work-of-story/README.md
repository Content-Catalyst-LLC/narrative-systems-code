# Myth, Ritual, and the Symbolic Work of Story

This companion directory supports the article **"Myth, Ritual, and the Symbolic Work of Story."**

The workflows operationalize myth and ritual analysis as a reviewable symbolic-story system. They include symbolic-function scoring, ritual-context diagnostics, interpretation-readiness review, ethical-risk scoring, power and authority review, cultural and access governance, markdown governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Myth and ritual are linked systems of symbolic narrative, embodied action, memory, authority, and social meaning. They can sustain identity, orientation, and transformation, but they can also justify exclusion, hierarchy, scapegoating, appropriation, or institutional power when interpreted or used irresponsibly.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Myth/ritual symbolic items, symbolic functions, ritual contexts, power/authority notes, ethical-risk, and governance datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for myth, ritual, symbolic work, and ethical-risk analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for myth, ritual, symbolic work, liminality, power, ethical risk, and responsible use

## Quick Start

From this article directory:

```bash
python3 python/run_myth_ritual_symbolic_canvas_audit.py
Rscript r/run_all_myth_ritual_symbolic_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
