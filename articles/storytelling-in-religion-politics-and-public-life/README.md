# Storytelling in Religion, Politics, and Public Life

This companion directory supports the article **Storytelling in Religion, Politics, and Public Life**.

The scaffold treats public storytelling as a governance problem: how do religious, political, civic, institutional, ritual, testimonial, media, and AI-mediated narratives shape memory, belonging, legitimacy, moral action, and public judgment?

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/public_story_governance_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable public-story governance review templates

## Quick Start

```bash
python3 python/run_public_story_governance_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_public_story_governance_workflows.R
```
