# Joseph Campbell and the Comparative Study of Myth

This companion directory supports the article **"Joseph Campbell and the Comparative Study of Myth."**

The workflows operationalize Campbell-informed comparative mythology as a reviewable interpretive system. They include comparative-pattern scoring, cultural-specificity diagnostics, generalization-risk review, interpretation-readiness checks, source-diversity review, counterexample tracking, ethical governance, markdown governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Joseph Campbell remains important because he helped modern audiences take myth seriously as symbolic story, but his comparative method must be used with care. Campbell is best treated as one interpretive lens, not a universal key. Responsible comparison must preserve source context, cultural specificity, language, ritual, oral performance, gender, power, counterexamples, and ethical limits.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Comparative myth claims, source contexts, pattern features, cultural-specificity notes, generalization-risk, and governance datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for Campbell-informed comparative mythology analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for Campbell context, comparative mythology, monomyth method, archetypal interpretation, critiques, responsible use, and governance notes

## Quick Start

From this article directory:

```bash
python3 python/run_comparative_myth_canvas_audit.py
Rscript r/run_all_comparative_myth_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
