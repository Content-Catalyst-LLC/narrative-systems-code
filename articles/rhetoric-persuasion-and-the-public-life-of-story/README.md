# Rhetoric, Persuasion, and the Public Life of Story

This companion directory supports the article **"Rhetoric, Persuasion, and the Public Life of Story."**

The workflows operationalize public story rhetoric as a governance-ready audit. They include ethos, pathos, and logos scoring; identification mapping; narrative persuasion force; evidence-support checks; public story risk scoring; emotional-pressure diagnostics; platform-circulation notes; rhetorical governance queues; and Catalyst Canvas-ready JSON exports.

## Core Claim

Stories have a public life. They persuade, legitimate, mobilize, accuse, defend, remember, and invite action. Public storytelling requires rhetorical analysis because narrative can support democratic reasoning, but it can also manipulate emotion, identity, memory, evidence, and public judgment.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Public story rhetoric datasets
- Rhetorical appeals, identification patterns, evidence support, public story risks, and governance notes
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for rhetorical story analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for ethos, pathos, logos, identification, persuasion risk, and responsible rhetorical use

## Quick Start

From this article directory:

```bash
python3 python/run_public_story_rhetoric_canvas_audit.py
Rscript r/run_all_public_story_rhetoric_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
