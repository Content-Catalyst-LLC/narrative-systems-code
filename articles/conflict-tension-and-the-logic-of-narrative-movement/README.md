# Conflict, Tension, and the Logic of Narrative Movement

This companion directory supports the article **"Conflict, Tension, and the Logic of Narrative Movement."**

The workflows operationalize conflict, tension, and narrative movement as a reviewable storytelling system. They include conflict-clarity scoring, tension-durability diagnostics, stakes visibility, narrative movement scoring, escalation and release analysis, ethical conflict-risk scoring, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Conflict is not just argument, violence, or opposition. In narrative, conflict becomes meaningful when it creates tension, pressure, movement, choice, consequence, uncertainty, and transformation. Tension sustains unresolved pressure over time, while narrative movement shows how pressure changes action, knowledge, relationship, value, or future possibility.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Conflict and tension datasets
- Conflict patterns, tension progressions, movement events, conflict risks, and governance notes
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for conflict and narrative movement analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for conflict versus tension, narrative movement, stakes, escalation, release, and ethical conflict governance

## Quick Start

From this article directory:

```bash
python3 python/run_conflict_tension_canvas_audit.py
Rscript r/run_all_conflict_tension_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
