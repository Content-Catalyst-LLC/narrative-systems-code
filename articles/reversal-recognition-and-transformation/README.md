# Reversal, Recognition, and Transformation

This companion directory supports the article **"Reversal, Recognition, and Transformation."**

The workflows operationalize reversal, recognition, and transformation as a reviewable storytelling system. They include reversal-integrity scoring, recognition-clarity diagnostics, transformation-depth analysis, retrospective coherence checks, recognition-risk scoring, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Reversal changes the direction of action. Recognition changes what is known. Transformation changes the condition of the story. Together, they reorganize meaning by turning situation, revealing hidden relation, and altering identity, responsibility, value, relationship, or future possibility.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Reversal and recognition datasets
- Reversal patterns, recognition events, transformation paths, recognition risks, and governance notes
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for reversal, recognition, and transformation analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for reversal, recognition, transformation, recognition risk, and responsible reveal design

## Quick Start

From this article directory:

```bash
python3 python/run_reversal_recognition_canvas_audit.py
Rscript r/run_all_reversal_recognition_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
