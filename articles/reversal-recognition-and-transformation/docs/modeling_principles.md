# Modeling Principles

## Reversal changes direction

A reversal turns the story's conditions. It should alter action, pressure, relation, power, expectation, or possibility.

## Recognition changes knowledge

Recognition reveals identity, cause, motive, evidence, memory, responsibility, or self-understanding.

## Transformation tests significance

A reveal matters when it changes action, identity, relationship, value, future possibility, or governance accountability.

## Surprise must be earned

A reversal should be surprising and retrospectively coherent.

## Recognition must be governed

Recognition can produce accountability, but it can also create false certainty, exposure risk, closure pressure, or evidence distortion.
