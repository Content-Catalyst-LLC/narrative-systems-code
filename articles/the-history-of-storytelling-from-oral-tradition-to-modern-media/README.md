# The History of Storytelling from Oral Tradition to Modern Media

This companion directory supports the article **"The History of Storytelling from Oral Tradition to Modern Media."**

The workflows operationalize storytelling history as a set of media transitions across oral tradition, myth, ritual, epic, manuscript culture, print, theater, novels, photography, film, broadcast media, digital platforms, games, and transmedia systems. They include media-transition scoring, transmission-strength diagnostics, preservation-risk audits, participation scoring, authority and access notes, historical governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

The history of storytelling is not a straight line from speech to writing to print to screen to platform. It is a layered history of media systems that overlap, preserve, transform, and recombine older story practices. Each medium changes memory, authority, circulation, participation, preservation, access, and power.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Story-media history datasets
- Oral tradition, manuscript, print, theater, film, broadcast, digital, and interactive media tables
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for historical storytelling analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for oral tradition, media transitions, archives, preservation risk, and responsible interpretation

## Quick Start

```bash
python3 python/run_story_media_history_canvas_audit.py
Rscript r/run_all_historical_storytelling_workflows.R
```

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
