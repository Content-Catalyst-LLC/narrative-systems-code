# Indigenous Storytelling, Place, and Relational Memory

This companion directory supports the article **Indigenous Storytelling, Place, and Relational Memory**.

The scaffold treats Indigenous storytelling as a governance and relational-accountability question, not as extractable narrative content. It includes diagnostics for place specificity, community authority, teller relationship, listener context, obligation visibility, protocol risk, place-memory strength, translation governance, digital-sovereignty risk, archive ethics, AI training risk, metadata flattening, and story-use review.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/indigenous_story_governance_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable Indigenous story governance review templates

## Quick Start

python3 python/run_indigenous_story_governance_audit.py
