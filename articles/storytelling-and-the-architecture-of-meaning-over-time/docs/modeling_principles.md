# Modeling Principles

## Meaning is temporal

Narrative meaning develops through origin, sequence, continuity, rupture, repetition, revision, and projection.

## Repetition is not proof

A repeated story can become durable without becoming accurate.

## Memory requires context

Preservation without context can create archive bias, false continuity, or decontextualized fragments.

## Drift is normal

Stories change meaning over time through repetition, omission, source aging, broken links, new audiences, and new evidence.

## Governance is necessary

Temporal storytelling requires source review, link maintenance, revision queues, context retention, and representation checks.
