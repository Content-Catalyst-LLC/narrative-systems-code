# Thresholds, Trials, and Transformative Ordeals

This companion directory supports the article **Thresholds, Trials, and Transformative Ordeals**.

The scaffold treats thresholds, trials, and ordeals as reviewable symbolic functions rather than decorative plot beats. It includes Catalyst Canvas-ready metadata, schemas, synthetic data, Python and R workflows, SQL schema files, governance queues, and documentation for auditing threshold strength, trial depth, ordeal transformation, ethical risk, source specificity, formula drift, and counterexamples.

## Quick Start

```bash
python3 python/run_threshold_ordeal_canvas_audit.py
Rscript r/run_all_threshold_ordeal_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
