# Proverb, Song, Chant, and Ritual Speech

This companion directory supports the article **"Proverb, Song, Chant, and Ritual Speech."**

The workflows operationalize compact oral forms as a reviewable storytelling and cultural-memory system. They include oral-form context scoring, sound-and-repetition diagnostics, ritual-authority review, translation-risk notes, archive-risk checks, platform-circulation review, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Proverbs, songs, chants, and ritual speech are not minor ornaments around narrative. They are compact, musical, rhythmic, and ceremonial forms of oral storytelling that carry memory, authority, identity, social judgment, sacred relation, and communal continuity.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Compact oral forms, oral-form contexts, sound/repetition features, ritual-authority notes, archive-risk, and governance datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for proverb, song, chant, and ritual speech analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for proverb, song, chant, ritual speech, translation/sound, archive risk, and responsible use

## Quick Start

```bash
python3 python/run_compact_oral_forms_canvas_audit.py
```
