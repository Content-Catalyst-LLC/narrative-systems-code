# The Hero's Journey: Departure, Initiation, and Return

This companion directory supports the article **The Hero's Journey: Departure, Initiation, and Return**.

The scaffold treats the hero's journey as a reviewable symbolic transformation model rather than a rigid formula. It includes Catalyst Canvas-ready metadata, schemas, synthetic data, Python and R workflows, SQL schema files, governance queues, and documentation for auditing departure, initiation, return, transformation depth, return responsibility, formula drift, and ethical risk.

## Quick Start

```bash
python3 python/run_heros_journey_canvas_audit.py
Rscript r/run_all_heros_journey_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
