# Oral Tradition, Performance, and Collective Memory

This companion directory supports the article **"Oral Tradition, Performance, and Collective Memory."**

The workflows operationalize oral tradition as a living narrative system. They include performance-context scoring, transmission-integrity diagnostics, collective-memory function analysis, variation mapping, archive-risk review, consent and ownership checks, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Oral tradition is not simply speech before writing. It is a living system of memory, performance, authority, variation, and cultural continuity. Oral narratives preserve knowledge, transmit values, organize identity, adapt to changing conditions, and keep collective memory alive through social and embodied practice.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Oral tradition, performance context, transmission, memory function, and archive-risk datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for oral tradition and performance analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for oral tradition, performance context, collective memory, archive risk, consent, ownership, and responsible preservation

## Quick Start

From this article directory:

```bash
python3 python/run_oral_tradition_canvas_audit.py
Rscript r/run_all_oral_tradition_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
