# Governance Notes

Review oral-tradition items when:
- performance context is weak
- transmission integrity is weak
- authority permission is unclear
- governance protocol is missing
- consent limits are high
- restricted knowledge is present
- exposure risk is high
- ownership risk is high
- extraction risk is high
- governance control is weak

Governance outputs should document:
- review owner
- tradition type
- performance context
- consent status
- authority and protocol
- access level
- archive restrictions
- reuse limits
- community review requirements
