# Responsible Use

These workflows are research and editorial scaffolds. They are not substitutes for community protocol, cultural expertise, legal review, trauma-informed review, archival ethics, or direct consent.

Use them to:
- audit performance context
- map transmission integrity
- identify collective memory functions
- review archive risk
- document consent and ownership issues
- create Catalyst Canvas-ready governance queues

Do not use them to automate decisions about sacred material, community ownership, restricted knowledge, cultural authority, or permission to publish.
