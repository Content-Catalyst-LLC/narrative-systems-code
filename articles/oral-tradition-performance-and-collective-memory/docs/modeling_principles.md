# Modeling Principles

## Oral tradition is living practice

Oral tradition is not merely unwritten text. It is a system of performance, memory, social authority, and cultural continuity.

## Performance is part of composition

Teller role, audience response, occasion, embodiment, setting, and cultural frame are part of meaning.

## Memory is social

Oral memory is distributed across voice, body, audience, place, ritual, repetition, and community recognition.

## Variation is not failure

Variation can be part of continuity. A living tradition changes while remaining recognizable.

## Preservation requires governance

Documentation should support community practice and protocols, not extract content or replace living transmission.
