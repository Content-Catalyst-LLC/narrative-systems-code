# Folktale Structure and Vladimir Propp’s Morphology

This companion directory supports the article **"Folktale Structure and Vladimir Propp’s Morphology."**

The workflows operationalize Proppian morphology as a reviewable storytelling system. They include function-coverage scoring, sequence-integrity diagnostics, sphere-of-action mapping, tale-variant comparison, morphology-context balance, formula-reduction risk, governance queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Vladimir Propp’s morphology of the folktale is a method for studying structured actions in wonder tales, not a universal formula for all stories. Propp’s functions and spheres of action help analysts compare tale movement, but they must be used with attention to corpus, performance, culture, language, variation, translation, and ethics.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Folktale morphology, Propp function, sphere-of-action, tale variant, morphology-risk, and governance datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for function mapping, sequence integrity, role distribution, and reduction-risk analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for Propp functions, spheres of action, sequence, variation, formula-reduction risk, and responsible morphology analysis

Rscript r/run_all_folktale_morphology_workflows.R
