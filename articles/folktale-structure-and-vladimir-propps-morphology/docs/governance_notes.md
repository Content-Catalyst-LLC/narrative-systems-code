# Governance Notes

Review folktale morphology items when:
- Propp is being used outside an appropriate corpus
- performance context is omitted
- cultural specificity is weak
- language and translation notes are missing
- variation is ignored
- source collection history is unclear
- tradition review is absent
- formula-reduction risk is high
- archive bias is high
- community sensitivity is high

Governance outputs should document:
- review owner
- tale type
- function coverage
- role mapping
- variant status
- source and translation notes
- cultural and performance context
- reduction risk
- revision recommendation
