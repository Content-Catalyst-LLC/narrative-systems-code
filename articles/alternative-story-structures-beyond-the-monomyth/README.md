# Alternative Story Structures Beyond the Monomyth

This companion directory supports the article **Alternative Story Structures Beyond the Monomyth**.

The scaffold treats alternative story structures as auditable interpretive models. It includes article-specific diagnostics for structural plurality, monomyth-overfit risk, alternative-structure readiness, medium fit, care narratives, archive and mosaic forms, interactive branching, cyclical structures, braided structures, relational forms, and AI-mediated structure governance.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/alternative_structure_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable structural-pluralism templates

## Quick Start

```bash
python3 python/run_alternative_structure_canvas_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_alternative_structure_workflows.R
```
