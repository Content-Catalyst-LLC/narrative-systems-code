# Modeling Principles

## Performance is part of the story

The teller, audience, occasion, place, voice, gesture, and interaction shape meaning.

## Memory is practiced

Oral memory is sustained by repetition, formula, sequence, audience recognition, community correction, and transmission pathways.

## Variation is evidence

Variation is not automatically error. It may indicate adaptation, skill, audience response, ritual restriction, or changing context.

## Documentation can distort

Transcripts, clips, and single recordings can flatten performance or overfix one version.

## Governance matters

Context, language notes, access protocol, source review, and community oversight should accompany variant analysis.
