# Responsible Use

These workflows are research and editorial scaffolds. They are not substitutes for community protocol, cultural review, language expertise, oral-history ethics, archival ethics, legal review, or direct consent.

Use them to:
- audit performance context
- map memory supports
- track variation responsibly
- identify archive and platform risks
- create Catalyst Canvas-ready governance queues

Do not use them to rank variants without source context, expose restricted material, or treat oral storytelling as inferior to writing.
