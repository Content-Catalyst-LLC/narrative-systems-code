# Governance Notes

Review oral storytelling variation items when:
- performance context is weak
- audience documentation is missing
- occasion context is missing
- language notes are weak
- access protocol is unclear
- community sensitivity is high
- platform circulation is involved
- archive risk is high
- variation is being treated as error without context
- one recording is being treated as definitive

Governance outputs should document:
- review owner
- storytelling context
- variation type
- source review
- access protocol
- archive risk
- revision recommendation
