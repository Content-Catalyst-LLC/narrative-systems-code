# Voice, Perspective, and Point of View

This companion directory supports the article **"Voice, Perspective, and Point of View."**

The workflows operationalize voice, perspective, and point of view as a reviewable storytelling system. They include voice-consistency scoring, perspective-access diagnostics, focalization clarity, reliability-risk checks, representation-gap review, institutional voice governance, and Catalyst Canvas-ready JSON exports.

## Core Claim

Voice, perspective, and point of view shape what kind of truth a story can offer. Voice concerns the manner and authority of telling. Perspective concerns the angle of perception and interpretation. Point of view concerns the structural access a story gives to knowledge, experience, memory, authority, and judgment.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Voice and perspective datasets
- Focalization maps, narrator access patterns, reliability risks, perspective gaps, and governance notes
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for voice, focalization, access, and reliability analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for voice, focalization, point-of-view access, reliability risk, representation gaps, and responsible perspective design

## Quick Start

From this article directory:

```bash
python3 python/run_voice_perspective_canvas_audit.py
Rscript r/run_all_voice_perspective_workflows.R
```

Outputs are written to:

outputs/json/
outputs/markdown/
