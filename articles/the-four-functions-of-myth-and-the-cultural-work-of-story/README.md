# The Four Functions of Myth and the Cultural Work of Story

This companion directory supports the article **The Four Functions of Myth and the Cultural Work of Story**.

The scaffold treats Campbell's four functions of myth as a reviewable cultural-work framework. It includes Catalyst Canvas-ready metadata, schemas, synthetic data, Python and R workflows, SQL schema files, governance queues, and documentation for auditing mystical, cosmological, sociological, and pedagogical functions; cultural work; sociological risk; institutional mythmaking; source context; counterexamples; and ethical accountability.

## Quick Start

```bash
python3 python/run_four_function_myth_canvas_audit.py
Rscript r/run_all_four_function_myth_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
