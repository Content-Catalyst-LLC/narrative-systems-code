# Moral Agency and the Stories We Tell About Ourselves

This companion directory supports the article **Moral Agency and the Stories We Tell About Ourselves**.

The scaffold treats moral self-narrative as an auditable responsibility, excuse-risk, and repair-readiness workflow. It includes article-specific diagnostics for action naming, intention distinction, consequence clarity, harm marking, repair orientation, other-person visibility, context overuse, intention shielding, victimhood shielding, blame shifting, growth substitution, harm minimization, and digital moral selfhood, plus the shared advanced Catalyst Canvas Python layer.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/moral_agency_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable moral-agency templates

## Quick Start

```bash
python3 python/run_moral_agency_canvas_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_moral_agency_workflows.R
```
