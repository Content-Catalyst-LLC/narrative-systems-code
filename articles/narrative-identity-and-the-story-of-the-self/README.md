# Narrative Identity and the Story of the Self

This companion directory supports the article **Narrative Identity and the Story of the Self**.

The scaffold treats narrative identity as an auditable self-story workflow. It includes article-specific diagnostics for memory continuity, temporal progression, agency, relational grounding, promise responsibility, revision readiness, trauma and silence governance, digital identity risk, and identity-story ethics, plus the shared advanced Catalyst Canvas Python layer.

## Quick Start

```bash
python3 python/run_narrative_identity_canvas_audit.py --article-root .
python3 python/run_catalyst_canvas_audit.py --article-root . --strict
Rscript r/run_all_narrative_identity_workflows.R
```
