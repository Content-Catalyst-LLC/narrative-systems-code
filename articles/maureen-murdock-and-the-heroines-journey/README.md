# Maureen Murdock and the Heroine's Journey

This companion directory supports the article **Maureen Murdock and the Heroine's Journey**.

The scaffold treats Murdock's heroine's journey as an auditable narrative-framework workflow. It includes article-specific diagnostics for heroine-journey alignment, framework risk, gender critique, integration quality, template forcing, gender essentialism, universal womanhood assumptions, psychological overreach, healing pressure, digital and AI-mediated use, plus the shared advanced Catalyst Canvas Python layer.

## What This Repository Adds

- Advanced additive `python/catalyst_canvas/` layer
- Article-specific `python/heroine_journey_canvas/` layer
- Typed dataclass models
- Config-driven scoring
- Validation
- Governance logic
- CSV, JSON, and markdown exporters
- CLI entrypoints
- Smoke tests
- Unit tests
- Canvas card generation
- Governance queue generation
- R diagnostics
- SQL schema and queries
- Documentation and reusable heroine-journey review templates

## Quick Start

```bash
python3 python/run_heroine_journey_canvas_audit.py
python3 python/run_catalyst_canvas_audit.py --article-root .
Rscript r/run_all_heroine_journey_workflows.R
```
