# Plot, Action, and Narrative Coherence

This companion directory supports the article **"Plot, Action, and Narrative Coherence."**

The workflows operationalize plot coherence as a reviewable storytelling system. They include plot-coherence scoring, action-dependency diagnostics, causal-linkage checks, motivation visibility, turning-point analysis, closure-pressure warnings, narrative coherence risk, ethical coherence governance, and Catalyst Canvas-ready JSON exports.

## Core Claim

Plot is not simply what happens. Plot is the arrangement of action into meaningful consequence. Narrative coherence is the relation among parts that allows a story to make sense without becoming simplistic, over-neat, or ethically distorted.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Plot coherence datasets
- Plot events, action dependencies, causal links, coherence risks, and governance notes
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for narrative coherence analysis
- HTML/CSS/PHP/Java scaffolds for Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for plot vs. events, action dependency, narrative coherence, coherence risk, and responsible story structure

## Quick Start

From this article directory:

```bash
python3 python/run_plot_coherence_canvas_audit.py
Rscript r/run_all_plot_coherence_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
