# What Is Storytelling?

This companion directory supports the article **"What Is Storytelling?"**

The workflows operationalize storytelling as a structured human system for organizing experience, memory, identity, culture, conflict, transformation, and meaning across time. They include narrative structure audits, event sequence modeling, character and agency mapping, motif inventories, conflict and transformation scoring, ethical representation checks, governance review queues, and Catalyst Canvas-ready JSON exports.

## Core Claim

Storytelling is not merely entertainment or decorative communication. It is a human system for arranging events, actors, settings, conflicts, values, memories, and transformations into meaningful form. A responsible storytelling analysis examines structure, context, voice, evidence, representation, power, and consequence.

## Repository Features

- Catalyst Canvas manifest and JSON schemas
- Narrative item, event, character, relationship, motif, and ethics datasets
- Python package-style audit workflow
- Base R diagnostic workflow
- SQL schema for narrative-system analysis
- HTML/CSS/PHP/Java scaffolds for article and Canvas integration
- Markdown governance queues
- JSON Canvas cards
- Synthetic datasets and generated output folders
- Documentation for narrative modeling, ethics, and responsible interpretation

## Quick Start

From this article directory:

```bash
python3 python/run_storytelling_canvas_audit.py
Rscript r/run_all_storytelling_workflows.R
```

Outputs are written to:

```text
outputs/tables/
outputs/json/
outputs/markdown/
outputs/figures/
```
